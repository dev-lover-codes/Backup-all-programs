# AI Voice Assistant and Agent

## Setup
The necessary dependencies have been installed.
- `google-genai`
- `pyautogui`

## How to Run
You can run the agent in two ways:

1. **Double-click** the `run_agent.bat` file in this directory.
2. **Terminal**: Open a terminal in this directory and run:
   ```bash
   python backend/agent.py
   ```

## Usage
- The agent will start and wait for your input.
- Type your request (e.g., "Open Calculator", "What is the screen size?").
- Type `exit` or `quit` to stop the agent.
