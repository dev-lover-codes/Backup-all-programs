# Quick Installation Guide

## Install PDF Pro Editor Extension

### Step 1: Open Extensions Page
1. Open Chrome or Edge browser
2. Navigate to `chrome://extensions/` (or `edge://extensions/` for Edge)
3. Enable **Developer mode** using the toggle in the top-right corner

### Step 2: Load the Extension
1. Click the **"Load unpacked"** button
2. Navigate to this folder: `e:\Programs\Extension\pdf editor`
3. Click **"Select Folder"**

## Troubleshooting

### Extension doesn't load
- Make sure you selected the correct folder (the one containing `manifest.json`)
- Check that Developer mode is enabled
- Try refreshing the extensions page and loading again

### Icons don't show
- The icons should be automatically loaded
- If not, check that the `icons` folder exists and contains the PNG files

### PDF doesn't open
- Make sure you're selecting a valid PDF file
- Check the browser console (F12) for any error messages
- Try a different PDF file
